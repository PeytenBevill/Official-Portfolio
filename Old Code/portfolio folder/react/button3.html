PetGuide

Introducing PetGuide, my React capstone project. PetGuide is a Full-Stack application, utilizing AWS to host my MySQL database, Vercel to host my backend, and Netlify to host my frontend. Users can create a new account or log into their existing account. Once in, PetGuide allows users to write custom pet care instructions modeled after FAQ pages for organization reasons. Other features include color-scheme options, CRUD operations to edit your instructions, and a sharable link to an un-editable page for your pet sitter to view!
